Spine快捷键

菜单

快捷键 说明
Cmd+Shift+N 新建项目骨架
Cmd+O 打开项目
Cmd+S 保存项目
Cmd+E 导出

通用

快捷键 说明
F1 显示工具提示信息
F2 重命名
H 选中目标的可见性
Cmd+Z 撤销
Cmd+Shift+z 取消撤销
Esc 取消选择

视图

快捷键 说明
鼠标右键按下拖动 平移
鼠标滚轮 缩放
Cmd+鼠标左键 复选
左键按下拖动 框选
Cmd+鼠标左键按下 框选且复选
D 预览动画

工具

快捷键 说明
C 旋转
V 移动
X 缩放
Z 倾斜(变形)
B 姿势
G 权重
N 创建骨骼
鼠标右键点击 最后的操作的两种工具之间切换
方向键 根据相对属性改变数值

旋转工具

快捷键 说明
Shift 锁定15°旋转

创建骨骼

快捷键 说明
Cmd 1.选中作为父级的骨骼
2.点选骨骼创建工具（或者按下快捷键N）
3.按住Cmd键点选图片，并在其上面创建骨骼。
4.然后松开Cmd键，创建完毕。
Option 选中骨骼并按住Option后重新创建，将更正之前绘制的不准确的骨骼。

打关键帧

快捷键 说明
K 为所有改变设置关键帧
L 为当前工具设置关键帧
Cmd+L 设置所有的关键帧
Cmd+Shift+L 设置所有关键帧，包括网格

设置模式

快捷键 说明
P 拾取父节点
('+’号) 在绘制顺序中向上移动1个单位
Shift + ('+’号) 在绘制顺序中向上移动5个单位
('-'号) 在绘制顺序中向下移动1个单位
Shift + ('-'号) 在绘制顺序中向下移动5个单位

动画模式

快捷键 说明
D 向前播放或停止
A 向后播放或停止
Q 回到动画开始的位置
E 跳转到动画结束的位置
W 下一个有关键帧的地方
S 上一个有关键帧的地方
R 下一帧
Shift+R 下10帧
F 上一帧
Shift+F 上10帧
Cmd+R 打开/关闭循环播放

层级树

快捷键 说明
鼠标左键双击 重命名
Cmod+鼠标左键 连续选中多个物体
鼠标右键点击右侧树状分支 展开/收起分支节点
